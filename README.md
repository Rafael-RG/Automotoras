# Apex Velocity - Premium Motoring Group

Una aplicación web premium para la gestión y exhibición de vehículos de alta gama.

## 🚀 Características

- **Catálogo de Inventario**: Visualiza vehículos premium con filtros avanzados
- **Localizador de Dealerships**: Encuentra concesionarios en la red global
- **Detalles de Producto**: Vistas detalladas de cada vehículo
- **Panel de Administración**: Gestión de flota e inventario
- **Portal de Registro**: Acceso para socios comerciales

## 🛠️ Tecnologías

- React 18
- React Router 6
- Tailwind CSS
- Google Fonts (Manrope & Inter)
- Material Symbols

## 📦 Instalación

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm start

# Compilar para producción
npm run build
```

## 🎯 Scripts Disponibles

- `npm start` - Inicia el servidor de desarrollo
- `npm run build` - Compila la aplicación para producción
- `npm test` - Ejecuta los tests

## 🌐 Navegación

- **/** - Página de inventario principal
- **/dealerships** - Localizador de concesionarios
- **/product/:id** - Detalle de vehículo específico
- **/register** - Registro de socios
- **/admin** - Panel de administración
